FAQ Accordion is a complete Frequently Asked Question Management WordPress plugin. It provides Simple FAQ custom post type With Setting panel.

== Description ==

It's easy to install the plugin and adding new questions and categories. After adding Frequently Asked Questions you can show the faq in your any Post, Page, and in your Widget are using FAQ shortcode., This is a simply customizable FAQ pluign for wordpress.  Don't believe on me try it fist. FAQ Responsive based on bootstrap framework and integrated with Font Awesome Icon.

https://www.youtube.com/watch?v=PZRKZjgC2_E&feature=youtu.be

You can easy sort your FAQ Accordion position using backend.  FAQ Responsive Plugin is built-in bootstrap. FAQ Accordion Design is Clean and retina ready.

> `[WPSM_FAQ id=xx]`

here xx is cpt id for FAQ

== Installation ==

1. Upload the entire `faq-responsive` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Faq Responsive menu to customize your FAQ plugin.
4. Copy your publish FAQ shortcode and then paste into your page or post to display your FAQ

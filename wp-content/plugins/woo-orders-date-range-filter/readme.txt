Easily filter the woocommerce orders with a date range.

== Description ==

Some times admin or shop-owner wants to filter orders day wise, month wise, year wise and by date range.With help of this plugin, Admin or Shop-owner can easily filter woocommerce orders day wise, month wise, year wise and by date range, Woocommerce > Orders.

== Installation ==
- Download the Plugin (in a zip file) on your system
- In the Admin Panel click on Plugin and select ‘Add New’ from the sub menu
- In the next screen Upload the plugin by browsing to the location where you have downloaded the plugin
- Activate the plugin through the 'Plugins' menu in WordPress

Or 

- Upload the woo-orders-date-range-filters (directory included) to the /wp-content/plugins/ directory
- Activate the plugin through the 'Plugins' menu in WordPress

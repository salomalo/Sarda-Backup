A) Installation - top

Extract woo-product-views.zip and upload to your wordpress plugin directory. Once uploaded visit your plugins page and activate "WooCommerce Product Views"

A settings page will be added under the main WooCommerce admin menu named "Product Views". When users start viewing products, a table will be populated with this information. If your website has high traffic, an easy to use live search has been implemented.
